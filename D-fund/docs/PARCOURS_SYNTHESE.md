# Parcours Utilisateurs - Synthèse

##   Approche

Basé sur l'analyse des données CSV Glide et la structure de l'application

## Parcours USER (Candidat)

### 1. Inscription / Connexion
- Créer un compte (email, password, nom)
- Choisir le type de profil (BtoC ou BtoB)
- Compléter le profil de base

### 2. Compléter son profil
**BtoC Profile** :
- Bio, compétences (tech/business)
- Industries d'intérêt
- Markets (régions)
- Langues
- Niveau de séniorité
- Disponibilité (remote, pays, régions)

**BtoB Profile** :
- Nom de l'entreprise
- Logo, header
- Description, industries
- Stage de développement

### 3. Explorer les opportunités
- Voir la liste des opportunités
- Filtrer par type, industrie, marché
- Rechercher
- Voir les détails d'une opportunité

### 4. Postuler à une opportunité
- Cliquer sur "Postuler"
- Remplir la candidature (Goal Letter)
- Sauvegarder en brouillon ou soumettre
- Suivre le statut (Draft → Submitted → Review → Success/Archived)

### 5. Gérer ses candidatures
- Voir la liste de ses candidatures
- Filtrer par statut
- Voir les détails et feedbacks
- Modifier les brouillons

### 6. Social
- Suivre/dé-suivre des users
- Like/Save des opportunités
- Voir les profils

### 7. Messages
- Envoyer/recevoir des messages privés
- Participer aux discussions publiques

## Parcours OWNER (Créateur d'opportunité)

### 1. Créer une opportunité

**Étape 1 : Sélection du type**
- Choisir parmi 18 types d'opportunités
- Exemples : Job Opportunity, Co-founder Opportunity, Event, etc.

**Étape 2 : Formulaire de création**
**Champs obligatoires** (déduits des CSV) :
- Nom de l'opportunité
- Description
- Type d'opportunité

**Champs optionnels** :
- Punchline
- Image / Background
- Localisation (City, Country, Remote)
- Dates (Start, End, Expiration)
- Industries
- Markets
- Tags
- URL
- Pricing (si applicable)
- Application Process (Paid, Check, Free Access)

**Options spéciales** :
- AI Generation (générer description automatiquement)
- Sauvegarder en DRAFT
- Publier directement (ACTIVE)

**Étape 3 : Publication**
- DRAFT → Sauvegardé, visible seulement par Owner
- ACTIVE → Publié, visible par tous
- PENDING → En attente modération (si applicable)

### 2. Gérer ses opportunités

**Liste des opportunités créées** :
- Voir toutes ses opportunités
- Filtrer par statut (Draft, Active, Archived, Closed)
- Voir les métriques (vues, candidatures, likes)

**Actions** :
- Éditer une opportunité
- Archiver/Fermer
- Supprimer (si Draft)
- Dupliquer
- Boost (premium)

### 3. Gérer les candidatures reçues

**Liste des candidatures** :
- Voir toutes les candidatures de ses opportunités
- Filtrer par statut (Draft, Pending, Active, Archived)
- Voir le nombre par opportunité

**Vue détaillée d'une candidature** :
- Profil du candidat
- Lettre de motivation (Goal Letter)
- Date de soumission
- Statut actuel

**Actions Owner** :
- Voir le profil complet du candidat
- Ajouter un feedback
- Changer le statut :
  - Submitted → Owner Review
  - Owner Review → Success (acceptée)
  - Owner Review → Archived (refusée)
- Envoyer un message au candidat

**Workflow de review** :
1. Notification "Nouvelle candidature"
2. Owner ouvre la candidature
3. Owner peut :
   - Voir le profil candidat
   - Ajouter feedback
   - Changer statut
4. Candidat reçoit notification

### 4. Dashboard Owner

**Métriques** :
- Nombre d'opportunités créées
- Nombre de candidatures reçues
- Opportunités actives
- Messages non lus

**Actions rapides** :
- Créer une opportunité
- Voir candidatures en attente

## Parcours ADMIN

### 1. Dashboard Admin

**Métriques globales** :
- Nombre total d'utilisateurs
- Nombre total d'opportunités
- Nombre total de candidatures
- Activité récente

### 2. Modération

**Opportunités** :
- Voir opportunités en PENDING
- Approuver → ACTIVE
- Rejeter (avec raison)
- Modifier
- Supprimer

**Utilisateurs** :
- Voir tous les utilisateurs
- Désactiver/Bannir
- Modifier profils
- Voir activité

**Contenu** :
- Modérer messages publics
- Modérer discussions
- Supprimer contenu inapproprié

### 3. Configuration

**Référentiels** :
- Gérer Industries
- Gérer Markets
- Gérer Features (types d'opportunités)
- Gérer Application Processes
