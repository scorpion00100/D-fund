## Analyse à effectuer

Pour chaque capture, identifier :

### 1. Structure et Layout
- [ ] Navigation (navbar) : structure, position, éléments
- [ ] Layout principal : colonnes, grilles, espacements
- [ ] Sidebar : présence, contenu, position

### 2. Couleurs
- [ ] Couleur principale (background)
- [ ] Couleur d'accent (boutons, liens)
- [ ] Couleurs de texte (titres, body, secondaires)
- [ ] Couleurs de statut (success, warning, error)

### 3. Typographie
- [ ] Police utilisée
- [ ] Tailles de texte (h1, h2, h3, body)
- [ ] Poids de police (bold, regular, light)

### 4. Composants
- [ ] Cartes d'opportunités : format, éléments affichés
- [ ] Boutons : style, taille, états
- [ ] Formulaires : champs, labels, validation
- [ ] Modals/Dialogs : structure, actions

### 5. Espacements
- [ ] Padding entre sections
- [ ] Marges entre éléments
- [ ] Espacement dans les grilles

### 6. Interactions visuelles
- [ ] États hover
- [ ] États actifs
- [ ] Loading states
- [ ] Animations/transitions
